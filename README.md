# Cisco Router-on-a-Stick Lab

## Overview

This project demonstrates how to configure inter-VLAN routing using a Cisco Router-on-a-Stick topology in Cisco Packet Tracer.

The network contains multiple VLANs, a Layer 2 switch, and a Cisco 2911 router that provides routing between VLANs using IEEE 802.1Q trunking.

---

## Network Topology

- Cisco 2911 Router
- Cisco 2960 Switch
- HR VLAN (10)
- IT VLAN (20)
- SALES VLAN (30)
- SERVER VLAN (40)

---

## VLAN Configuration

| VLAN | Name | Network |
|------|------|---------|
|10|HR|192.168.10.0/24|
|20|IT|192.168.20.0/24|
|30|SALES|192.168.30.0/24|
|40|SERVERS|192.168.40.0/24|
